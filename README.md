# DRK Utils

Utilities of the government of the Democratic Republic of Kinda.

## Features

* ID generation
	* Generate many IDs with 1 command
	* Supports prefixes
	* Supports checksums
* Quick KMIN link

More features will be added in the future.

## Usage

```bash
git clone https://github.com/AlexPi314/drk-utils.git
cd drk-utils
npm install
```
